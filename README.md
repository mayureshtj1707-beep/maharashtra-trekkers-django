# Maharashtra Trekkers Django Project

A web application built with **Django** for showcasing trekking destinations and organizing trekking events in Maharashtra, India.

---

## Features

- List and explore trekking destinations in Maharashtra.
- View detailed information about each trek.
- User registration and authentication.
- Admin panel to manage treks and users.
- Responsive and user-friendly interface.

---

## Technologies Used

- **Backend:** Django, Python
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** django
- **Version Control:** Git, GitHub
